# ScheduleApp

Приложение расписания для студентов и преподавателей вуза МТУСИ в рамках проекта для дисциплины "Программная инженерия".

## Запуск

1. Убедитесь, что установлен Node.js.
2. Перейти в терминале в папку `Project\ScheduleApp`
3. Запуск сервера:

```bash
npm run start
```

Откройте приложение по адресу `http://localhost:3000`.

## Профили

- Студент: только просмотр пар и комментариев.
- Преподаватель: добавление, редактирование и отмена пар.
- Администратор: те же права, что и у преподавателя.
